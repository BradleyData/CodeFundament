An article is independent, self-contained content. It should maked sense on its own and it should be possible to distribute it independently from the rest of the site.
An aside is some content loosely related to the page content.
A summary for details. Details represent a widget from which the user can obtain additional information or controls.
A div specifies a division or section in a document.
A nav defines a section of navigation links.

A p defines a paragraph; (from the Greek paragraphos, "to write beside" or "written beside") a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

A section of a document such as a header, footer, etc.
A form defines a section with controls for user input.
A legend assigns a caption to a fieldset. A fieldset specifies a set of related form fields.
